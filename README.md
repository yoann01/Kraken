Kraken
======
Kraken provides a flexible, customizable and extendable framework that leverages Fabric Engineâ€™s Splice & Canvas API's. Kraken deploys across 3D applications and because of its use of Fabric Engine's Splice & Canvas API's, maintains functionality no matter what DCC is used to build a rig.
